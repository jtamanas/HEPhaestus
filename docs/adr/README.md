# Architecture Decision Records (ADRs)

This directory contains Architecture Decision Records for the HEPhaestus
marketplace. Each ADR documents a significant design or scope decision, the
context that motivated it, the decision taken, its consequences, and
alternatives that were considered and rejected.

## Index

| ADR | Title | Status |
|-----|-------|--------|
| [0001-cosmology-scope-gate.md](0001-cosmology-scope-gate.md) | Cosmology scope gate for HEPhaestus | Accepted (2026-05-02) |

New ADRs should follow the template used in `0001-cosmology-scope-gate.md`:
**Status**, **Context**, **Decision**, **Consequences**, **Alternatives
Rejected** — in that order. ADR numbering is sequential; the filename prefix
is zero-padded to four digits.
