# Overlay Attributions: lz_xenonnt_pandax4t_2024

This overlay bundle covers three post-2022 direct-detection experiments for
use with DDCalc 2.2.0. Individual experiment attributions are in the
per-experiment NOTICE files.

**Status: DEFERRED to v1.1.** The overlay plumbing is in place but the
efficiency tables and Fortran modules are stubs pending v1.1 authorship.

---

## Experiments

### LZ WS2024
- **Collaboration:** LUX-ZEPLIN (LZ)
- **Result:** arXiv:2410.17036 (2024 WIMP-search data release)
- **Primary reference:** Phys. Rev. Lett. (PRL) 131, 041002 (2023); arXiv:2207.03764
- **NOTICE:** `LZ_WS2024.NOTICE`

### XENONnT 2023
- **Collaboration:** XENON
- **Result:** arXiv:2303.14729
- **Primary reference:** Phys. Rev. Lett. (PRL) 131, 041003 (2023)
- **NOTICE:** `XENONnT_2023.NOTICE`

### PandaX-4T 2021
- **Collaboration:** PandaX-4T
- **Result:** arXiv:2107.13438
- **Primary reference:** Phys. Rev. Lett. (PRL) 127, 261802 (2021)
- **NOTICE:** `PandaX_4T_2021.NOTICE`
