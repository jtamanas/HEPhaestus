**Adopted BP triplets (plan convention):**
- BP1: (M_R=42.0, mu_S=0.05, M_N=42.0) GeV → |X_1| = 42.0 GeV ✓
- BP2: (M_R=59.0, mu_S=0.05, M_N=59.0) GeV → |X_1| = 59.0 GeV ✓
- BP3: (M_R=61.0, mu_S=0.05, M_N=61.0) GeV → |X_1| = 61.0 GeV ✓

Verification: with the 3×3 mass matrix where `m_E = M_R`, `mu_1 = mu_2 = mu_S`, `m_N = M_N`, the lightest eigenvalue has `|X_1| = M_R` exactly when `M_R = M_N` (confirmed numerically).

**Deviation from plan placeholders: NONE** — the values `{42, 59, 61}` are confirmed.
