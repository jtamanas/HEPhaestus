# Plot A/B Test Eval Suite

Compares plots generated by Claude Code agents WITH vs WITHOUT the hephaestus
design system (color palettes, typography, layout rules, plotting skills).

## Prerequisites

- Python 3.10+ with numpy, scipy, matplotlib
- `claude` CLI installed and authenticated
- `--dangerously-skip-permissions` available (or `--permission-mode bypassPermissions`)
- ~$24 API budget ceiling (12 cases × 2 agents × $1.00 max each; actual spend typically lower)

## Usage

```bash
# Full run: generate data → spawn agents → build voting page → open in browser
python run_eval.py

# Just generate synthetic data (no agent runs)
python run_eval.py --data-only

# Just rebuild the voting page from existing outputs
python run_eval.py --view-only

# Limit concurrent agent pairs (default: 4)
python run_eval.py --max-parallel 2
```

## How It Works

1. Generates synthetic HEP data for 12 plot types (invariant mass, pT spectra, exclusion contours, etc.)
2. For each case, spawns two `claude` CLI agents in parallel:
   - **With skills**: gets the design system, style sheets, and plotting skill injected into its prompt
   - **Without skills**: gets only a generic "make a publication-quality HEP plot" instruction
3. Both agents run in isolated `/tmp` directories with `--bare` to prevent CLAUDE.md discovery
4. Collects output PNGs and builds a self-contained HTML voting page
5. Opens the page in your browser — vote blind on which plot looks better
6. Download results as JSON when done

## What It Tests

| Case | Plot Type | Design Rules Exercised |
|------|-----------|----------------------|
| z_mass_peak | 1D histogram | Signal/background colors |
| pt_spectrum | Log-scale spectrum | Line weight hierarchy |
| stacked_backgrounds | Stacked fills | Color ordering, legend |
| exclusion_contour | Brazil bands | Fill opacity, contour styling |
| theory_data_ratio | Ratio panel | Uncertainty bands, error bars |
| parameter_scan_2d | 2D heatmap | Colormap, contour lines |
| running_couplings | Multi-line | Color escalation |
| angular_distribution | Line plot | LaTeX typography |
| cross_section_vs_sqrt_s | Points + band | Error bars + theory band |
| missing_et | Stacked + log | Log y-axis, overlay |
| branching_ratios | Many curves | Direct labeling |
| multi_panel_comparison | Small multiples | Panel spacing |
